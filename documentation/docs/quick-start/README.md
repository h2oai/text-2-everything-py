## Before you begin

### All guides
* Text2Everything API endpoint (default usually works)
* Platform access token
* H2O GPTe API key
* Workspace with Workspace Owner permission
* Python 3.9+

### [Advanced integration only]
* H2O Drive account
* H2O GPTe UI access
* Snowflake database (optional)

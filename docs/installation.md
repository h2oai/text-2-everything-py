---
title: Installation
---

For now, install locally:

```bash
pip install -e .
```

Future (PyPI):
```bash
pip install text2everything-sdk
```

Optional extras:
```bash
pip install -e ".[dev,integrations]"
```

Troubleshooting:
- Ensure Python 3.9+
- If behind a proxy, set `HTTP_PROXY`/`HTTPS_PROXY`
